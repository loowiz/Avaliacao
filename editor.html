Dica: Fórmulas Matemáticas (LaTeX)

Você pode usar códigos LaTeX para exibir fórmulas matemáticas nas perguntas ou alternativas. O quiz usa a biblioteca KaTeX.

  • Na mesma linha: Use $ ... $. Ex: $E=mc^2$
  • Bloco centralizado: Use $$ ... $$.
⚠️ IMPORTANTE: Como este gerador cria um arquivo JavaScript, você deve usar DUAS barras invertidas (\\) para os comandos LaTeX.

Exemplo Fração: $\\frac{1}{2}$ (correto) em vez de $\frac{1}{2}$ (errado).
Exemplo Ohm: $10\\Omega$
